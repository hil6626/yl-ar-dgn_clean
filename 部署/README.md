```

├── 🖥️ YL-monitor/          # 统一监控平台 (端口5500)

├── ⚙️ AR-backend/           # 核心业务后端 (端口5501)
│   └── 视频/音频/人脸合成服务
├── 👤 user/                 # 用户GUI前端 (端口5502)
│   └── PyQt5 AR合成界面
├── 📜 rules/                # 五层规则架构
├── 🔧 scripts/              # 统一脚本入口
```
